2.3. Описание работы с модулем распознавания 

Созданный модуль распознавания предназначен для решения задач определения классов морских объектов вооружения по существующим РЛИ космических средств наблюдения. Алгоритм разработан с целью сжатия данных и передачи их на наземные приемные пункты в интересах оперативного обеспечения радиолокационной информацией потребителей ВМФ. Для достижения высокой вероятности распознавания необходимо обеспечить правильность моделирования получения опорных бинарных РЛИ в различных условиях наблюдения.

Главная проблема, возникающая при решении задачи правильного распознавания - это искажение реальных размеров морских ОВВТ вследствие специфичности построения их РЛИ в координатах наклонная дальность-азимут и не достаточно высокое пространственное разрешение.

Анализ искажений, возникающих на полученных с помощью ПК  опорных РЛИ, с учетом известных закономерностей проявления эффектов образования радиолокационных теней, наложений и укорочений, позволяет сделать вывод о правильности использованных методов моделирования. Известно, что эффекты образования радиолокационных теней наиболее отчетливо проявляются при почти настильных углах визирования и ослабевают при увеличении угла места.

Проявление эффектов укорочения и наложения имеет обратный характер: существенно искажают форму объекта при больших углах места. При ухудшении разрешения точность геометрических параметров получаемых РЛИ падает и эффекты образования радиолокационных теней, наложений и укорочений становятся менее заметными. Сравнивая опорные РЛИ для различных углов места при одинаковых азимутах обнаруживается эффект укорочения, который при уменьшении угла места меньше искажает длину и ширину объекта. Наложение наиболее сильно искажает ширину объекта и проявляется при больших углах места.

Правильность и точность получаемых опорных изображений позволит добиться распознавания морских ОВВТ с большей вероятностью и обеспечить потребителей ВМФ достоверной информацией.

Классификация по функциональному назначению (21 класс морских ОВВТ), заложенная в базу исходных данных, построена следующим образом. Каждому классу соответствует двумерный чертеж объекта в трех ортогональных проекциях с размерами близкими к математическому ожиданию длины, ширины, высоты объектов этого класса, а также дисперсия геометрических параметров. Чтобы определить какой геометрический признак ОВВТ является наиболее приемлемым для проведения распознавания, необходимо исследовать их на устойчивость к искажениям  вследствие эффектов теней, наложений, укорочений, при изменении условий радиолокационного наблюдения.

При классификации по функциональному назначению наблюдается пересечение классов по длинам. С ухудшением разрешающей способности дисперсия внутри класса возрастает, и, следовательно, неопределенность отнесения объекта по его длине к тому или иному классу увеличивается. При полном пересечении двух классов один из них исчезает, а остается тот, у кого математическое ожидание длины больше и количество классов уменьшается. Поэтому, при построение банка опорных РЛИ, различным разрешениям будет соответствовать разное количество классов.

При распознавание некоторых классов, например, крейсер и десантный корабль по первичному признаку - длине ОВВТ - дешифрировать нельзя из-за сходства их изображений для некоторых ракурсов и всех разрешений. Поэтому, для исключения неоднозначности необходимо привлекать дополнительные признаки морских ОВВТ.

При использование координат объекта, целесообразно вторичные признаки основывать на местоположение ОВВТ относительно других обнаруженных морских ОВВТ на кадре РЛИ, то есть привлекать структурные связи ордера кораблей. Анализ ордеров кораблей показывает, что крейсер используется в боевом охранении в составе АУГ на строго определенном расстоянии, где десантный корабль оказаться не может. Таким образом, вторичным признаком распознавания для неразделяемых классов морских ОВВТ по длине целесообразно использовать расстояние до главной цели ордера, определяемое в процессе вычисления координат для всех ОВВТ.

Диапазон изменения по углу азимута выбирается в пределах от 0° до 359° (выбор всего диапазон необходим из-за ассиметрии авианосцев относительно центральной оси). Диапазон по углу места - определяется ТТХ КРСА. Выбор дискретности изменения углов зависит от разрешающей способности и определяется динамикой изменения геометрических параметров РЛИ объекта при изменении условий радиолокационного наблюдения.

Для произведение функции распознавания пользователь необходимо сформировать бинарный образ РЛИ ОВВТ.  Форма загрузки нового поискового бинарного образа РЛИ позволяет добавить бинарный образ для поиска совпадений в БД ОРЛИ (рис. 11).

 

 

Рис. 11 – Загрузка бинарного РЛИ

Форма поиска позволяет выполнить поиск соответствий загруженного бинарного ОРЛИ и найти совпадающие по характеристикам морские ОВВТ из БД ОРЛИ (рис. 12).

 

 

Рис. 12 – Поиск опорного РЛИ в БД

 Таким образом, модуль распознавания  ПК ФРБРИ ОВВТ позволяет пользователю выполнять процедуры поиска морских ОВВТ на первичных снимках радиолокационного наблюдения за земной поверхностью, сохранять результаты поиска и производить обмен данными с другими пользователями.

 

 

3. Выводы

Наиболее устойчивым признаком разделения классов морских ОВВТ и их опорных РЛИ является длина, поэтому при распознавании необходимо определить параметры реального РЛИ ОВВТ. Для разделения близких классов, например, крейсеров, транспортных и десантных кораблей, имеющих схожие размеры, требуется привлекать вторичные признаки - параметры взаимного расположения в типовом ордере.

Проведенный анализ показал, что применение данного программного комплекса актуально для обработки РЛИ в АСУ военного назначения и имеет следующие преимущества:

- обеспечивается совместимость с существующими компонентами АСУ ВС РФ;

- обеспечивается решение задач распознавания морских ОВВТ;

- обеспечивается повышение оперативности  обработки данных РЛИ.
